5. transparent means :

allowing light to pass through so that objects behind can be distinctly seen. (bahasa : transparan, jernih, tembus cahaya)

6. opaque means :

not able to be seen through; not transparent. (bahasa : buram, tidak tembus cahaya)

7. transparent object examples :

- water

- clear glass lightbulb

- clear window pane

- clear plastic wrap

8. opaque object examples :

- cardboard

- book

- bricks

- buildings
