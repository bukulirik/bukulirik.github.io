17. Ardy, bidu, Dan coky mendapat tugas ronda dikampungnya. Ardi bertugas Setiap 2hari sekali, bidu bertugas Setiap 5 hari sekali, Dan coky bertugas Setiap 3 hari sekali. Jika tanggal 26 February 2019 mereka akan bertugas bersama, maka tanggal berapakah mereka akan bertugas lagi?


kpk dari 2,3,5 adalah 30

di 2019 bulan februari ada tanggal sebanyak 28 hari

jadi 26 februari + 30= 56 februari -28= 28 maret

jadi mereka akan bertugas bersama sama lagi tanggal 28 maret 2019

semoga beemanfaat

Diketahui =

Ardi bertugas setiap 2 hari sekali.

Bidu bertugas setiap 5 hari sekali.

Coki bertugas setiap 3 hari sekali.

Pada tanggal 26 Februari mereka bertugas bersama.

Ditanya = Tanggal berapakah mereka akan bertugas lagi ???

Jawab =

Ardi = 2 x 1

Bidu = 5 x 1

Coki = 3 x 1

Kpk = 2 x 3 x 5

= 30

Bertugas lagi = 30 + Tanggal 26

= 56 - banyak tanggal di bulan Februari 2019

= 56 - 28

= 28 Maret 2019

Jadi,mereka akan bertugas lagi pada tanggal 28 Maret 2019.

Semoga Bermanfaat :-)
