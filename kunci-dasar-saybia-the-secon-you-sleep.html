[tex] \sf\underline { Present \: Simple \: Tense } [/tex]

Present Simple Tense digunakan untuk menyatakan :

- Kejadian yang terjadi sekarang

- Kebenaran umum

- Aksi kebiasaan

[tex] \\ [/tex]

[tex] \sf\underline { Rumus \: Present \: Simple \: Tense } [/tex]

1. Verbal, Subject I, You, We, They

(+) Subject + Verb 1 + Object

(-) Subject + "don't" + Verb 1 + Object

(?) "Do" + Subject + Verb 1 + Object

[tex] \\ [/tex]

2. Verbal, Untuk Subject He, She, It

(+) Subject + Verb 1 + "s/es" + Object

(-) Subject + "doesn't" + Verb 1 + Object

(?) "Does" + Subject + Verb 1 + Object

[tex] \\ [/tex]

3. Nominal, Subject I

(+) Subject + "am" + Adjective/Noun/Adverb

(-) Subject + "am" + "not" + Adjective/Noun/Adverb

(?) "Am" + Subject + Adjective/Noun/Adverb + ?

[tex] \\ [/tex]

4. Nominal, Subject You, We, They

(+) Subject + "are" + Adjective/Noun/Adverb

(-) Subject + "are" + "not" + Adjective/Noun/Adverb

(?) "Are" + Subject + Adjective/Noun/Adverb + ?

[tex] \\ [/tex]

5. Nominal, Subject He, She, It

(+) Subject + "is" + Adjective/Noun/Adverb

(-) Subject + "is" + "not" + Adjective/Noun/Adverb

(?) "Is" + Subject + Adjective/Noun/Adverb + ?

34. If I were you I would sleep early. The fact is...


The fact is...

Answer:I'm not you so I don't sleep early.

Discussion:

Menandai kalimat kondisional menjadi kalimat berdasarkan fakta dengan merubah struktur kalimatnya menjadi bertentangan dengan kalimat kondisional tersebut.

Kalimat kondisional meliputi tiga tipe dan di atas merupakan tipe dua—menyatakan suatu keberadaan/kegiatan di masa lalu

(if-clause — nominal form) + verbal form

if + simple past + past future

if + S + was/were + O + S + auxiliary + V1 + ...

Dengan menggunakan to be was/were diamati dari subyeknya.Subyek dan obyek berupa noun atau pronoun—bisa juga dilengkapi adjective.

Untuk merubahnya berdasarkan kenyataan, sama halnya merubah kalimat positif menjadi negatif atau sebaliknya dan ubah tense-nya menjadi simple present.

S + is/am/are + not + O + so + S + do/does + not + V1 + ...

Jika diperhatikan pada kalimatnya:

If I were you I would sleep early.

if: ifsubject (S): Iwas/were: were (bisa was hanya dianggap formal)object (O): yousubject (S): Iauxiliary: wouldverb-infinitive (V1): sleepadverb: early

Jika diubah menjadi kalimat kenyataan bisa dua kemungkinan:

I'm not you so I don't sleep early.

atau:

I don't sleep early because I'm not you.

Note:Untuk pernyataan pertama, didahului nominal form maka menyatakan dirinya bukan diri yang ditunjuk—gunakan so sebagai penghantar verbal form. (saya bukan Anda maka saya...)Untuk pernyataan kedua, didahului verbal form maka menyatakan dirinya tidak melakukan seperti orang itu—gunakan because sebagai penghantar nominal form. (saya (tidak)... karena saya bukan Anda)
