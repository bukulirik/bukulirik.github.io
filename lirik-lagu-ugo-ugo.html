18. pidato ugo diarani.....​


Wangsuli pitakon

Pidato ugo diarani sesorah.

Pembahasan

Pidhato yaiku ngandharake panemu utawa gagasan marang wong akeh utawa marang tamu undangan. Kagunakane kanggo sarana micara sing wis kasusun kanthi tata cara urutan lan paugeran sing becik.

Perangan teks Pidhato :

1. P@mbuka Pidhato yaiku salam pakurmatan sing sepisan marang para tamu undangan sing kepareng rawuh.

2. Purwaka yaiku atur Pamuji syukur marang Gusti Allah sarta atur panuwun marang para tamu undangan.

3. Isi utawa surasa basa yaiku sakabehe bab sing diandharake marang para tamu undangan.

4. Dudutan utawa kesimpulan yaiku ringkesan isi sing wigati utawa intine sing diandharake.

5. Pangarep arep yaiku pituduh lan pangajab sing diandharake marang para tamu undangan.

6. Wasana basa utawa panutup yaiku atur panuwun sarta Nyuwun pangapura menawa ana kekurangan utawa kaluputan olehe ngandharake.

Jinising Pidhato ana 2 :

1. Pidhato resmi, yaiku pidhato sing nggunakake basa resmi miturut EYD utawa panulisan sing disampurnakaken.

➡️ Tuladha :

Pidhato presidenPidhato sambutan upacara bendera

2. Pidhato ora resmi, yaiku pidhato sing nggunakake basa padinan utawa basa sing digunakake ing saben dinane lan ora kudu nggawe basa sing resmi nanging tetep miturut adicara sing ana ing pidhato.

➡️ Tuladha :

Pidhato ambal warsaPidhato syukuran ngresmekake usaha utawa liyane.
