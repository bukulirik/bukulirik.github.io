7. Penjelasan dari 1. Front-End Developer 2. Back-End Developer adalah ?


Penjelasan:

1. Front end developer

Front end developer adalah orang yang berperan mengembangkan tampilan situs atau aplikasi melalui HTML, CSS, dan JavaScript.

Jadi Front end developer mengubah tampilan menjadi lebih menarik. Front end developer disebut juga client-side

2. Back end developer

adalah orang yang bekerja dibelakang situs agar situs dapat tetap berjalan.

Back end developer sangat penting, Ia bertugas untuk memastikan agar apa yang dibuat oleh front end developer atau sistem dan server di balik situs atau aplikasi dapat bekerja. Posisi ini biasa disebut server-side.

Front-End

Bagian front-end dari sebuah website adalah bagian yang langsung dilihat oleh user. User juga bisa langsung berinteraksi pada bagian ini. Bagian ini dibangun menggunakan HTML, CSS, dan JavaScript.
